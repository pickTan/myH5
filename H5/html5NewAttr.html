manifest

< html lang="zh-CN" manifest="cache.manifest" >表示此文件需离线下载保存，一般用于webapp

target

<base target="_blank" >base表示个目录，target表示此页面所有href都用target方法打开

defer and async

<script defer src="_blank" >defer 表示下载完全部资源并编译后再执行此资源

<script async src="_blank" >async 表示下载此资源后马上执行，同时并不停止浏览器解析下面的数据

media 、download、ping、 hreflang and ref

<a media="handhelp" > 手持 表示会对手持设备进行优化

<a media="tv" >电视 表示会对电视进行优化

<a hreflang="zh" >表示超链接所拥有的语言

<a ref="external" >表示超链接是内部还是外部超链接 external 外部

<a ping="url" > 表示超链接点击时会发送一个请求到此路径

<a href="url" download="url" > 表示超链接点击时下载文件，一些模版的下载可以前端直接做

<ol start="50" reversed < 表示从50开始 倒叙排列

  1. Coffee
  2. Tea
  3. Milk
